# ASHFIRE

A real time chat application with login and room access feature.

<img src="client/Ashfire.PNG">

## Introduction

Interactive chat Application

## Features

- room access
- spam detection
- emoji support
- new user notification
- Neumorphism

## Installation

- ```console
  $ cd client
  $ npm install

  ```

-

```console
 $ cd server
 $ npm install

```

## Start

- To start react framework

  ```console
  $ cd client
  $ npm start

  ```

- To start server
  ```console
   $ cd server
   $ npm start

  ```
